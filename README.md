# React-final-project

To Run Locally-

Run Backend-
$ cd backend
$ npm install
$ npm start

Run Frontend-
# open new terminal
$ cd frontend
$ npm install
$ npm start


Seed Users and Products (backend)-
Run this on browser: http://localhost:5000/api/seed
(Will returns admin email and password and 6 sample products).

Admin Login-
Run http://localhost:3000/signin

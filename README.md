## To Run Locally-

### Run Backend-

```
$ cd our-website
$ cd backend
$ npm i
$ npm start
```

### Run Frontend-

```
open new terminal

$ cd our-website
$ cd frontend
$ npm i
$ npm start
```

Seed Users and Products (backend)-
Run this on browser: http://localhost:5000/api/seed
(Will returns admin email, password and sample products).

Admin Login-
Run http://localhost:3000/signin
